<p style="color:green># Login Form Project</p>

A web project with a login form styled using HTML and CSS.

## Installation / How to Run

1.Download or unzip the project folder.
2.Open 'login.html' in zour browser.

## Usage 

- Enter a username and password in the form (for demonstration purposes, any values work;
  no backend is connected).
- The **Login** button will submit the form on the page itself (no real authentication).

## Features

- Stylish login form with CSS.
- Responsive design for different screen sizes (if implemented).
